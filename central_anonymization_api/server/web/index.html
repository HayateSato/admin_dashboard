How It Works

The ECG signal is split into non-overlapping Tumbling Windows of T seconds each. Within each window, values are sorted and grouped using a pre-built generalization hierarchy. The algorithm promotes each value to a coarser range until each group contains at least k identical values — satisfying k-anonymity. The midpoint of each range is then used as the published value (mean imputation).

Trade-off: a higher k provides stronger privacy but distorts the signal more. A longer Tumbling Window T preserves more signal shape because more values per window gives the algorithm more flexibility — enabling finer-grained anonymization ranges.
